# Application: db_react_portfolio

## Application Description: 
An updated React version of my personal portfolio. Featuring React, Bootstrap and EmailJS. 
Additional React packages include: 
- react-hook-form
- react-icons
- react-popupbox
- react-scripts
- react-scroll
- react-typed
